# ComfyUI-TextJoinZveroboy

A ComfyUI custom node: **Text Join Zveroboy**.

Features:
- Starts with 2 text inputs (text_1, text_2).
- Automatically adds more inputs (text_3, text_4, ...) when all current inputs are connected.
- Joins any non-empty inputs; does NOT require text_1 to be connected.

## Install
Clone or download into:
`ComfyUI/custom_nodes/ComfyUI-TextJoinZveroboy`

Restart ComfyUI.

## Node
Category: Zveroboy/Text
